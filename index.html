Reddit launches new tools to improve user engagement

Reddit has introduced new tools to help users follow community rules and track content performance, aiming to boost engagement on the platform. The update comes after a slowdown in user growth due to Google’s algorithm changes, though traffic from the search engine has since recovered. Among the new features is a ‘rules check’ tool, currently being tested on smartphones, which helps users comply with subreddit guidelines.
